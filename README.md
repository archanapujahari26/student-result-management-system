🎓 Student Result Management System

A Python + SQLite based Student Result Management System that stores student records, calculates percentage and grades automatically, and helps analyze student performance.

⸻

🚀 Features
	•	Add student details (name & marks)
	•	Automatic percentage calculation
	•	Grade generation (A, B, C, Fail)
	•	Store data using SQLite database
	•	View all student records
	•	Simple command-line interface (CLI)

⸻

🛠 Tech Stack
	•	Programming Language: Python
	•	Database: SQLite
	•	Concepts Used: CRUD operations, database management, basic data analysis
